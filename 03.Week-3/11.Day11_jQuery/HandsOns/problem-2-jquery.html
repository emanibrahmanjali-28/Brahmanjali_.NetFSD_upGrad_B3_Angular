1. What is jQuery Lib?
Ans:- jQuery is a fast, small, and feature-rich JavaScript library that simplifies HTML DOM traversal, event handling, animations, and Ajax for rapid web development.


2. What are Toggle Methods in jQuery?
Ans:- The core jQuery toggle methods are used to switch between two states (like showing/hiding elements or adding/removing a class) and include .toggle(), .slideToggle(), .fadeToggle(), and .toggleClass().


3. What is DOM Traversing?
Ans:- DOM Traversal is the process of navigating up, down, or sideways through the HTML tree structure (nodes) using JavaScript to locate and manipulate specific elements.


4.How do you include jQuery in a web page?
Ans:- You can include the jQuery library via a script tag, either by linking to a local file or using a Content Delivery Network CDN like Google or Microsoft.


5.Is jQuery a replacement for JavaScript?
Ans:- No, jQuery is a library built on top of JavaScript. It uses JavaScript's methods and properties to provide a simpler, more concise syntax and handle cross-browser compatibility issues, but it is not a separate language.
